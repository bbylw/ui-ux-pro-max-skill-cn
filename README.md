## 🚀 快速开始

### 本地预览

```bash
# 克隆仓库
git clone <your-repo-url>
cd mv

# 使用任意静态服务器预览
npx serve ./

# 或者直接在浏览器中打开
open index.html
```

### 在线部署

本项目可以直接部署到任何静态托管服务：

- **Vercel**: 拖拽文件夹或连接 Git 仓库
- **Netlify**: 拖拽部署或 Git 集成
- **GitHub Pages**: 推送到 `gh-pages` 分支
- **Cloudflare Pages**: 连接 Git 仓库自动部署
